# AI Research Paper Summarizer

An AI-powered web application that helps users understand complex AI research papers through customizable explanations.

Users can select a research paper, choose the explanation style and length, and receive a well-structured explanation generated using Google's Gemini model through LangChain.

## Features

- Explain popular AI research papers
- Multiple explanation styles
  - Beginner-Friendly
  - Technical
  - Code-Oriented
  - Mathematical
- Adjustable explanation length
- Interactive Streamlit interface
- Dynamic Prompt Engineering using LangChain

## Tech Stack

- Python
- Streamlit
- LangChain
- Google Gemini
- Prompt Engineering
