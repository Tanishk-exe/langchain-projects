# AI Product Review Analyzer

An AI-powered application that transforms long product reviews into structured insights using Google's Gemini model and LangChain.

The application extracts important information such as sentiment, rating, summary, key points, pros, and cons using Structured Output powered by Pydantic.

## Features

- Automatic review summarization
- Sentiment Analysis
- Rating Extraction (1–5)
- Pros & Cons Detection
- Key Point Extraction
- Structured Output using Pydantic
- Interactive Streamlit interface

## Tech Stack

- Python
- Streamlit
- LangChain
- Google Gemini
- Pydantic
- Structured Output
